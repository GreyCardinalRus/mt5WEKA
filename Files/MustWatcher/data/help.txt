=====  Список команд  =====
 1) help - справка бота
 2) status - посмотреть текущее состояние торгового счета
 3) report - получить отчет по электронной почте
 4) myemail - посмотреть текущий установленный email
 5) myemail;<your_email> - установить адрес электронной почты для получения отчетов
 6) spamon - включение авто-аповещения
 7) spamoff - отключение авто-аповещения
=======================
*все команды начинаются с спецсимвола "!" (например '!help')
