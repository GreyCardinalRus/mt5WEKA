MustWatcher v1.16
----------------------------------
Для запуска необходим "Microsoft .NET Framework 3.5" 


I.Смысл утилиты

Программа предназначена для отправки содержимого txt-файлов в виде icq сообщения на uin с которого был отправлен соответствующий запрос.


II.Принцип работы

Программа представляет собой по сути icqbot’а.
Боту выделяется uin от которого будут посылаться сообщения.
Задается каталог txt-файлы которого будут доступны для запроса.
После подключения бот находится в режиме ожидания сообщений, которые анализирует на соответствие командам.
Список доступных команд вызывается '!help' (находится в data '..\help_admin.txt')


III.Группы доступа

В боте существуют всего 3 группы пользователей:
1)Админ – задается в боте при настройке в соответствующем поле и имеет полный доступ
2)Зарегистрированные пользователи – добавляются админом и имеют доступ к ограниченному списку команд, который также вызывается '!help' (находится в data '..\help.txt')
3)Все остальные – не имеют доступ к вводу команд, могут только получать скриптовые ответы бота (которые добавляются двумя предыдущими группами) на свои сообщения.


IV.Доступ к содержимому txt-файлов

Для того чтобы получить содержимое текстового файла по аське, необходимо сформировать список в вкладке 'Watcher' при помощи кнопки 'Add'. Командой для вызова является имя файла без расширения (например: отправив боту по аське команду '!status' получаем содержимое файла 'status.txt').


V.Отчеты по email

Бот может отправлять детализированые отчеты на указанный email получив соответствующую команду по аське. Для этого предварительно нужно добавить файл с данными в список при помощи кнопки 'Add'. Командой для составления и отправки отчета является имя файла данных без расширения (например: отправив команду '!report' бот сформирует отчет из файла 'report.txt').


VI.Обратная связь

В боте реализована функция обратной связи, которая позволяет записывать любой текст в заданный файл. Для этого в вкладке 'Feedback' необходимо нажать 'Add', после чего:
1)Добавить имя команды по получению которой будет производится запись. 
2)Указать записываемый текст.
3)Задать файл записи.
4)Посылаем по аське команду боту в формате: !<имя добавленной команды>*
*Если необходимо записать в файл дополнительный текст, то его следует добавить к запросу через ";". 
Полный формат получится следующим: !<имя добавленной команды>;<дополнительный текст>
